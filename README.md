# SK Social Post Builder (SPB)

A professional WordPress plugin that generates ready-to-share social media messages from WordPress posts in one click.

## Features

- **Dashboard**: Track key stats, view generated counts, and access quick actions.
- **Create Post Page**: Interactive multi-step wizard to search/filter posts, arrange order with drag & drop sorting, select platforms (WhatsApp, Telegram, Facebook, LinkedIn, X, Custom), configure format options, and generate compilation message.
- **History Module**: Track previously compiled messages, view statistics, and support quick copying/duplication of messages.
- **Templates Module**: Manage default built-in templates or build custom message templates with placeholder support.
- **Settings**: Adjust fallback defaults, autodelete logs, and configure UI preferences.

## How to Use

1. **Select Posts**: Navigate to the **Create Post** screen. Step 1 allows you to search and filter posts by keyword, category, tag, author, or custom date range. Select the checkbox next to the posts you want to compile.
2. **Arrange Order**: Go to Step 2 to view your selected posts list. Use the drag-and-drop handles to reorder the posts exactly how you want them formatted in your final message.
3. **Choose Template**: In Step 3, choose your target template platform (e.g. WhatsApp, Telegram, Facebook, LinkedIn, X, or a Custom template).
4. **Display Options**: Check or uncheck formatting details in Step 4, including list numbering, emojis, default footer, website URL, hashtags, and duplicate filtering.
5. **Generate & Share**: Go to Step 5, click **Generate**, click **Copy to Clipboard** to copy the compiled text, and paste it directly into WhatsApp, Telegram, or any other social channel.
